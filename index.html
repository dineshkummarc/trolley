Trolley - Box2D utility

Please see this jsfiddle for example starting point.

All the examples on this page can be found here.

Simple

 trolley.init();
 trolley.body(0, 0, true).box(15, 1);
 trolley.body(5, 1).box(4, 2);
 trolley.body(7, 3).box(2, 2);
 trolley.body(5, 3).box(2, 2);
 trolley.body(6, 8).box(2, 2);
editable

Chaining

 trolley.init();
 trolley.body(0, 0, true).box(15, 1);
 trolley.body(1, 1).box(1, 3);
 trolley.body(1, 10).box(2, 2, 2, 2).circle(2).circle(0, 4, 2).circle(4, 0, 2).circle(4, 4, 2);
editable

How shapes inside a body can be chained together

Options

 trolley.init();
 trolley.body(0, 0, true).box(15, 1);
 for (var i = 1; i < 8; i += 2) {
     trolley.body(5, i).box(2, 2);
 }
 trolley.body(4.5, 15).box(2, 2, {
     density: 100
 });

Body options

 trolley.init();

 trolley.body(5, 1).box(2, 2);

     fixedRotation: true
 }).box(2, 2, {
